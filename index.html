03 Mes projets

01

Sokoban / Projet BUT

Jeu de logique dans lequel le joueur doit pousser des caisses sur des cibles, réalisé en langage C, afficher sur le terminal.

02

Problème du cavalier / Projet BUT

Problème de parcours du cavalier sur un échiquier 8 X 8 devant passé par toutes les cases une unique fois, résolu par un algorithmes de recherche en profondeur pour optimiser le programme, reealisé en python.

03

Picross / Projet BUT

Jeu de logique dans lequel le joueur doit remplir une grille de cases en fonction des indices donnés, réalisé en python, dans le but d'optimiser un programme.

04

Site Web / Projet BUT

Conception et développement d'un site web responsive, réalisé en HTML, CSS pour un site de tourisme, pour le Royaume-Unis.

05

Gestion de Projet / Projet BUT

Imagination de toutes les étapes, en passant par la planification des fonctionnalités au maquettage, pour la conception d'une application, afin d'aider une boulangerie à optimiser sa gestion de stock et de vente.

06

Installation d'un poste pour le développement / Projet BUT

Installation et configuration d'un poste de développement, réalisé en bash et PHP, pour faciliter la conversion de différents types de fichiers reçus, pour la création d'un site web.

07

Base de Données de parcoursup 2022/ Projet BUT

Mise en place d'une base de données pour stocker et gérer les informations des candidats sur parcoursup 2022, sur SQLWorbench.
